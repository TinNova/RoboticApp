# Robotic Investment Club App

Created an app for stock investment clubs, it allows clubs to keep track of their stocks, view research done my members and discuss the clubs activities.



## Technologies Used

- Fragments
- RecyclerView
- Volley
- IntentService
- SQL
- Firebase Analytics & Ads
- Material Design
